# zlib

zlib 是 DEFLATE 生态中最常见的工程库之一。它不是最新的算法，但兼容性非常强。

## 核心思想

DEFLATE 结合 LZ77 字典匹配和 Huffman 编码。zlib 提供稳定、广泛可用的接口。

## 工程取舍

| 维度 | 特点 |
| --- | --- |
| 压缩速度 | 中等 |
| 解压速度 | 中等到较快 |
| 压缩率 | 中等 |
| 内存 | 较低 |
| 流式 | 支持 |

## 适用场景

- 需要广泛兼容的文件或网络格式
- gzip/zlib 生态集成
- 老系统和跨平台组件
