# Brotli

Brotli 常用于 Web 内容压缩，尤其适合静态文本资源。它在压缩率上通常优于 gzip，但压缩端成本更高。

## 核心思想

Brotli 结合字典、上下文建模和熵编码，并内置静态字典来改善常见文本内容的压缩效果。

## 工程取舍

| 维度 | 特点 |
| --- | --- |
| 压缩速度 | 高级别较慢 |
| 解压速度 | 较快 |
| 压缩率 | 文本和 Web 资源表现好 |
| 内存 | 依质量级别变化 |
| 流式 | 支持 |

## 适用场景

- Web 静态资源
- 文本内容分发
- 对传输体积敏感的 HTTP 场景
