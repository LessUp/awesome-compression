# LZMA

LZMA 常见于 xz/7z 生态，追求较高压缩率，代价是压缩速度和内存占用通常高于轻量算法。

## 核心思想

LZMA 使用大窗口字典匹配和范围编码，并通过上下文建模提升压缩率。

## 工程取舍

| 维度 | 特点 |
| --- | --- |
| 压缩速度 | 通常较慢 |
| 解压速度 | 中等到较快 |
| 压缩率 | 很高 |
| 内存 | 可较高 |
| 流式 | liblzma 支持流式接口 |

## 适用场景

- 软件包发布
- 长期归档
- 对压缩率更敏感而非压缩速度的场景
