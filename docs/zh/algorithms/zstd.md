# ZSTD

Zstandard 是一个面向通用场景的现代无损压缩算法，目标是在高压缩率和高速度之间提供宽范围可调节的平衡。

## 核心思想

ZSTD 结合字典匹配和熵编码。它支持多个压缩级别，低级别适合实时数据管道，高级别适合归档。

## 工程取舍

| 维度 | 特点 |
| --- | --- |
| 压缩速度 | 从很快到较慢可调 |
| 解压速度 | 通常很快 |
| 压缩率 | 通用数据上表现稳定 |
| 内存 | 随压缩级别增加 |
| 流式 | 支持 |

## 适用场景

- 日志压缩
- 数据管道
- 本地缓存
- 可调压缩级别的归档任务
