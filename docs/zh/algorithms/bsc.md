# BSC

BSC 是一个适合学习块排序和上下文建模思想的压缩库。它的价值不仅在于工程使用，也在于帮助理解“变换 + 编码”的组合方式。

## 核心思想

BSC 通常围绕块级处理、排序变换和后续编码展开，让相似上下文聚集后再压缩。

## 工程取舍

| 维度 | 特点 |
| --- | --- |
| 压缩速度 | 依配置变化 |
| 解压速度 | 依配置变化 |
| 压缩率 | 对部分数据较有竞争力 |
| 内存 | 块大小越大占用越高 |
| 流式 | 更偏块处理 |

## 适用场景

- 学习块排序压缩
- 对特定数据集做实验性比较
- 离线批处理
