# 如何选择算法

没有一个压缩算法适合所有场景。选型时先明确约束，再比较算法。

## 常见约束

| 约束 | 优先考虑 |
| --- | --- |
| 实时传输 | LZ4、ZSTD 低压缩级别 |
| 长期归档 | LZMA、ZSTD 高压缩级别、Brotli |
| Web 内容 | Brotli、zlib |
| 日志与数据管道 | ZSTD、LZ4 |
| 兼容性优先 | zlib |
| 学习块排序思想 | BSC |

## 判断顺序

1. 解压速度是否是硬约束。
2. 压缩端是否允许更高 CPU 和内存成本。
3. 格式是否需要被其他工具直接读取。
4. 数据是否有大量重复、结构化字段或自然语言文本。
5. 部署环境是否容易引入第三方库。
